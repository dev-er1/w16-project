# Навигация по этой папке (example-langs)

**Здесь находятся экспериментальные языки программирования, которые используют W16 runtime как бэкенд.**

## Структура проекта

### [karbon](./karbon/)
Экспериментальный язык программирования.

**Статус:** Разработка в процессе (приоритет отдается W16 runtime)

**Документация:** [karbon/KARBON.md](./karbon/KARBON.md)

**Характеристики:**
- Собственный синтаксис
- Лексер и парсер на Rust
- Интеграция с W16 runtime

### [w16-cc](./w16-cc/)
Экспериментальный компилятор языка **C** (стандарт C11) с бэкендом W16.

**Статус:** Активная разработка

**Документация:** [w16-cc/W16-CC.md](./w16-cc/W16-CC.md)

**Ключевые компоненты:**
- **Типы данных:**
  - Целочисленные: `int`, `short`, `long`, `long long` (знаковые и беззнаковые)
  - С плавающей точкой: `float`, `double`, `long double` (80-bit F80)
  - Комплексные типы (C99/C11): `_Complex float`, `_Complex double`, `_Complex long double`
  
- **F80 реализация:**
  - Собственный тип `F80` для поддержки 80-битных чисел (long double в C)
  - Операции: сложение, вычитание, умножение, деление
  - Конвертация из f64 (double)
  - Нормализация чисел и работа с субнормализованными значениями
  - Полный набор тестов в `tests/f80_tests.rs`

## Примечания

- Некоторые компоненты `w16-cc` взяли из `karbon`
- Оба языка находятся на ранних стадиях разработки и являются экспериментальными
- Основной приоритет разработки — оптимизация W16 runtime
