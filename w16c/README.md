# w16c

Здесь будет рассказано про этот крейт.

## О крейте
w16c — экспериментальный крейт, который должен генерировать исполняемые файлы. Пока что он экспериментальный,
потому что:

- На самом деле он генерирует объектные файлы, а для сборки испоняемых файлов нужен линковщик.

- Для генерации исполняемых файлов он использует `link.exe` от Visual Studio. То есть чтобы генерировать исполняемые файлы,
нужно иметь установленный Visua Studio.

### Вкратце о том, как он работает
Здесь используется библиотеку `cranelift-object`, для создания объектных файлов.
Она позволяет создавать объектные файлы для разных платформ. И код был взят из `w16-core\src\jit\jit_compiler.rs`, но были изменены
некоторые части, чтобы он создавал объектные файлы, а не исполнял их.
