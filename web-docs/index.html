W16

A lightweight runtime for creating programming languages. Build a frontend, emit W16 HIR or MIR, then run through a shared bytecode VM or Cranelift JIT.

Quickstart

Run HIR from Rust

[dependencies]
w16-lib = "0.1.1"
use w16_lib::run_hir_text;

fn main() {
    let src = r#"
module main {
  fn @main() -> u64 {
    print(42)
    return 42
  }
}
"#;

    let result = run_hir_text(src).unwrap();
    println!("{}", result.r0());
}

Architecture

Layered by design

Frontend

Parses a source language and emits W16 HIR or MIR. WCC is the main showcase frontend.

HIR

High-level typed IR for functions, variables, expressions, control flow, and calls.

MIR

SSA-style middle IR for verification, analysis, optimization, and bytecode lowering.

Runtime

Register bytecode executed by the VM or compiled in memory through the Cranelift JIT.
